# daoboard

> VueJS + Wilddog 实时显示daocloud构建状态等webhook消息

![](http://ww3.sinaimg.cn/large/006tNc79jw1fakkgo5urkj31401z4qc9.jpg)

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```
